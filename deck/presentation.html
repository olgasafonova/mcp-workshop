What is MCP?

MCP is a protocol: a shared language that lets AI assistants talk to your apps.

Before USB

Every device had its own cable

Printers, cameras, phones, keyboards: all different connectors

After USB

One standard, everything connects

MCP is USB for AI

Using MCP Effectively

You're giving AI a toolbox. Be specific about which app to use.

Good Prompts

  • "Use the wiki server to search for authentication docs"
  • "Create a Miro sticky note saying 'Review API design'"
  • "Use Playwright to take a screenshot of the login page"

Less Effective

  • "Find some info about authentication"
  • "Make a note somewhere"
  • "Open a website"

Architecture Overview

server/
├── main.go              # Entry point, server setup
├── tools/
│   ├── definitions.go  # What can it do? (schemas)
│   └── handlers.go     # How does it do it? (logic)
└── client/
    └── client.go       # API client for the app

The minimal server needs:

1. A tool definition (what can it do?)

2. A handler (how does it do it?)

3. A client (how does it talk to the app?)

Production Patterns

What I added as servers matured:

Error Handling
Retries, graceful failures
Auth Flows
OAuth, API keys, tokens
Caching

Rate Limiting
Respect API limits

From Miro (77 tools): Bulk operations, complex schemas, audit logging

From GLEIF (4 tools): Clean separation, minimal dependencies, no auth

Getting Started

Fork an Existing Server

  • GLEIF — Simplest (no auth, 4 tools)
  • Miro — Most complete (all patterns, 77 tools)

Use an SDK

# Go
go get github.com/modelcontextprotocol/go-sdk

# TypeScript
npm install @modelcontextprotocol/sdk

MCP Specification: modelcontextprotocol.io

17 / 19

Key Takeaways

  • Explorers: MCP lets AI take action, not just answer questions. It works with cloud apps AND local apps on your machine.
  • Users: Setup is simple. Download, configure, restart. Start with GLEIF (cloud, no auth) or Playwright (local, visual).
  • Builders: Start small. My first server had 5 tools. By the fourth, I had production patterns figured out.
