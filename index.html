GLEIF + Claude Desktop

Give Claude the ability to look up company information

1 Download

Download from GitHub

Choose your file:

Mac M1/M2/M3/M4: gleif-mcp-server-darwin-arm64.mcpb Mac Intel: gleif-mcp-server-darwin-amd64.mcpb Windows: gleif-mcp-server-windows-amd64.mcpb

2 Unblock the file

Open Terminal (Cmd+Space, type "Terminal") and paste this entire line:

chmod +x ~/Downloads/gleif-mcp-server-darwin-arm64.mcpb && xattr -d com.apple.quarantine ~/Downloads/gleif-mcp-server-darwin-arm64.mcpb

Press Enter. No output = it worked.

If you see a Windows SmartScreen warning:

Click "More info" → "Run anyway"

The file is safe — it's just not signed.

3 Configure Claude Desktop

1. In Finder, press Cmd+Shift+G

2. Paste: ~/Library/Application Support/Claude/

3. Open claude_desktop_config.json (create if missing)

4. Paste this (replace YOUR_USERNAME):

{ "mcpServers": { "gleif": { "command": "/Users/YOUR_USERNAME/Downloads/gleif-mcp-server-darwin-arm64.mcpb" } } }
Find your username: Open Terminal, type whoami, press Enter.

1. Press Win+R

2. Paste: %APPDATA%\Claude

3. Open claude_desktop_config.json (create if missing)

4. Paste this (replace YOUR_USERNAME):

{ "mcpServers": { "gleif": { "command": "C:\\Users\\YOUR_USERNAME\\Downloads\\gleif-mcp-server-windows-amd64.mcpb" } } }
Important: Use double backslashes \\ in Windows paths!

4 Restart & Test

1. Quit Claude completely (Mac: Cmd+Q | Windows: right-click taskbar → Close)

2. Open Claude Desktop

3. Type: "Look up Apple's LEI"

You should see Claude call the GLEIF tool and return company data. Done!

Troubleshooting

Problem Solution
Server not available Check path for typos. Restart Claude.
Mac security warning Run the Terminal command from Step 2
Windows SmartScreen Click "More info" → "Run anyway"
JSON error Check for missing commas. Validate at jsonlint.com
